# 🎮 Tic-Tac-Toe Game (C# & Windows Forms)

A classic **Tic-Tac-Toe** game built with **C#** and **Windows Forms** (Desktop Application). This project focuses on UI/UX customization and clean logic implementation as part of my Software Engineering journey.

---

## 🚀 Features
* **Modern UI:** Custom-designed icons and buttons for a smooth experience.
* **Game Logic:** Accurate win/draw detection and game reset functionality.
* **Dynamic Visuals:** Real-time turn indicator and score tracking.

## 🛠️ Built With
* **Language:** C#
* **Framework:** .NET Framework 4.8
* **Technology:** Windows Forms (WinForms)
